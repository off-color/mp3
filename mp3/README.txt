Программа умеет читать тэги из mp3 файла, а также всю информацию о  фреймах.
Параметры запуска: [-t] [-f] [-p] [-s FILENAME] MP3File.
Ключ -t для вывода информации о id3 тэгах.
Ключ -f для вывода информации о фреймах.
Ключ -w для вывода информации об аудиоданных в виде волны.
Для данной опции нужен установленный ffmpeg.
Ключ -s служит для сохранения обложки альбома в файл.
Ключ -p для воспроизведения.
Для более подробной информации о том, как правильно ввводить аргументы
в программу, запустите справку командой: './mp3.py -h'.
Примеры запуска: 
	python3 mp3.py -i MySong.mp3
